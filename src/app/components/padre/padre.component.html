Soy Padre


Se trata de un ejemplo para comprobar como enviar y recibir datos de un componente (Padre) a otro (Hijo).
- Para pasar datos al Hijo en el Padre se asigna una variable a una propiedad Input del Hijo.
- Para recibir datos del Hijo, el Hijo tiene que emitir un evento (output) y el padre controlar dicho evento.
